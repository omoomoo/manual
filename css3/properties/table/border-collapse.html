border-collapse

语法

border-collapse：collapse | separate | inherit

取值

取值 说明
collapse 设置表格的边框合并成一条（border-spacing和empty-cells属性将会被忽略）。
separate 设置表格的边框分开（border-spacing和empty-cells属性将会被不会忽略）。
inherit 继承。

说明

border-collapse属性设置边框是否合并成一条。

默认值 separate
继承 yes
版本 css2
Javascript脚本 object.style.borderCollapse="collapse"

兼容性

IE FireFox Chrome Safari Opera
         

Note: IE6-IE9不支持inherit。

Note: If a !DOCTYPE is not specified, the border-collapse property can produce unexpected results.

示例

   
   

border-collapse:
inherit
collapse

separate
