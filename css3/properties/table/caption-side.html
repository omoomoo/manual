caption-side

语法

dcaption-side：top | bottom

取值

取值 说明
top Puts the caption above the table. This is default
bottom Puts the caption below the table
inherit Specifies that the value of the caption-side property should be inherited from the parent element

说明

默认值 top
继承 yes
版本 css2
Javascript脚本 object.style.captionSide="bottom"

兼容性

IE FireFox Chrome Safari Opera
         

Note: IE8 supports the caption-side property only if a !DOCTYPE is specified.
