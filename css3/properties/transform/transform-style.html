transform

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

transform：none|transform-functions;

取值

transform-functions取值 说明
none 无变换
matrix(n,n,n,n,n,n) 定义一个2D矩阵变换，包含6个参数。
matrix3d
(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) 		定义一个2D矩阵变换，包含16个参数。
translate(x,y) 定义一个2D平移，x为X轴，y为Y轴。
translate3d(x,y,z) 定义一个3D平移，x为X轴，y为Y轴，z为Z轴（相当于垂直屏幕）。
translateX(x) 定义一个X轴的平移。
translateY(y) 定义一个Y轴的平移。
translateZ(z) 定义一个Z轴的平移。
scale(x,y) 定义一个2D缩放。x为X轴，y为Y轴。
scale3d(x,y,z) 定义一个3D缩放。x为X轴，y为Y轴，z为Z轴。
scaleX(x) 定义一个X轴的缩放。
scaleY(y) 定义一个Y轴的缩放。
scaleZ(z) 定义一个Z轴的缩放。
rotate(angle) 定义一个2D旋转，旋转的角度作为参数。
rotate3d(x,y,z,angle) 定义一个3D旋转，？？？？
rotateX(angle) 定义一个3D旋转，？？？？
rotateY(angle) 定义一个3D旋转，？？？？
rotateZ(angle) 定义一个3D旋转，？？？？
skew(x-angle,y-angle) 定义一个2D扭曲，x-angle为水平方向的旋转角度，y-angle为垂直方向的旋转角度。
skewX(angle) 定义一个水平方向旋转的2D扭曲。
skewY(angle) 定义一个垂直方向旋转的2D扭曲。
perspective(n) 给一个变换的元素定义一个透视图。

说明

transform属性给元素提供了2D和3D的变换效果。这些效果包括旋转、矩阵变换、缩放、扭曲等。

默认值 none
继承 否
版本 CSS3
Javascript脚本 object.style.transform="rotate(7deg)"

兼容性

The transform property is not supported in any browser.

Internet Explorer supports an alternative, the -ms-transform property (2D transforms only).

Firefox supports an alternative, the -moz-transform property (2D transforms only).

Opera supports an alternative, the -o-transform property (2D transforms only).

Safari and Chrome support an alternative, the -webkit-transform property (3D and 2D transforms).
