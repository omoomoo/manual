transform-origin

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

transform： ( percentage | length | left | center | right ) ( percentage | length | top | center | bottom )? length?

取值

transform-functions取值 说明
none 无变换
matrix(n,n,n,n,n,n) 定义一个2D矩阵变换，包含6个参数。
translate(x,y) 定义一个2D平移，x为X轴，y为Y轴。
translate3d(x,y,z) 定义一个3D平移，x为X轴，y为Y轴，z为Z轴（相当于垂直屏幕）。

说明

改变元素基点

他的主要作用就是让我们在进行transform动作之前可以改变元素的基点位置，因为元素默认基点是元素中心点，但有时候我们需要在不同的位置对元素进行这些操作，那么我们就可以使用transform-origin来对元素进行基点位置改变，使元素基点达到你需要的基点位置。

transform-origin(X,Y)：其中X和Y的值可以是百分值,em,px，其中X也可以是字符参数值left,center,right；Y和X一样除了百分值外还可以设置字符值top,center,bottom，这个看上去有点像我们background-position设置一样；一般我们直接使用百分值,em,px设置值方便些。

默认值 50% 50% 0
继承 否
版本 CSS3
Javascript脚本 object.style.transformOrigin="20% 40% 0px"

兼容性

Internet Explorer supports an alternative, the -ms-transform-origin property, but only for 2D-transforms.

Firefox supports an alternative, the -moz-transform-origin property, but only for 2D-transforms.

Opera supports an alternative, the -o-transform-origin property, but only for 2D-transforms.

Safari and Chrome supports an alternative, the -webkit-transform-origin property, on both 2D- and 3D-transfoms.

示例

<img src="../../images/transform.jpg" style="-webkit-transform:rotate(10deg)"/>

 

/*正常情况下元素的基点在元素的正中间，如图中的蓝点位置，元素的变形将围绕该蓝点展开。*/

<img src="../../images/transform.jpg" style="-webkit-transform:rotate(10deg); -webkit-transform-origin:left top;"/>

 

/*-webkit-transform-origin:left top;将元素的基点移到左上角，rotate(10deg)的选择围绕左上角展开，如图所示*/

<img src="../../images/transform.jpg" style="-webkit-transform:rotate(10deg); -webkit-transform-origin:left bottom;"/>

 

/*-webkit-transform-origin:left top;将元素的基点移到左下角，rotate(10deg)的选择围绕左下角展开，如图所示*/
