transition

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

transition： property duration timing-function delay

取值

取值 说明
transition-property 设置对象中的参与过渡的属性。
transition-duration 设置对象过渡的持续时间。
transition-timing-function 设置对象中过渡的动画类型。
transition-delay 设置对象延迟过渡的时间。

说明

设置元素的过渡效果。

过渡效果将会在元素被hover时发生。这意味着，如果你要使ID为mydiv的元素产生过渡效果，你要有两步（如第一个实例）。

默认值 all 0 ease 0
继承 否
版本 CSS3
Javascript脚本 object.style.transition="width 2s"

 

兼容性

示例

该元素的hover效果会动哦！

<div id="transition"></div>

#transition {

/*第一步:transition指定过渡的属性为transform，调速函数为ease-in-out 过渡属性为1s 延时0s*/
-webkit-transition:-webkit-transform ease-in-out 1s 0s;
}

/*第二步:指定元素的hover，并为第一步指定的过渡属性transform赋值，这里指定元素旋转360度，那么该元素将会使用调数函数ease-in-out，用1s旋转360度，延迟1s执行*/
#transition:hover {
-webkit-transform:rotate(360deg);
}

该元素的hover效果会动哦！

#demo2 {
-webkit-transition:-webkit-transform ease 3s 0s;
}

#demo2:hover {
-webkit-transform:rotate3d(0,1,0,360deg);
}
