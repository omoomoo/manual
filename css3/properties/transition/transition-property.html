transition-property

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

transition-property：all | none | property

取值

取值 说明
all 所有可以进行过渡的css属性。
none 不指定过渡的css属性。
property 指定要进行过渡的css属性。

说明

transition-property属性指定要进行过渡的css属性。

** 过度效果通常情况下是在当元素的hover被触发时发生。

默认值 all
继承 否
版本 CSS3
Javascript脚本 object.style.transitionProperty="width"
