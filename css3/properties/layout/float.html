float

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

float：nont | left | rigth

默认值：none

取值

none：对象不浮动

left：对象向左浮动。

right：对象向右浮动。

说明

With CSS float, an element can be pushed to the left or right, allowing other elements to wrap around it.

Float is very often used for images, but it is also useful when working with layouts.

兼容性

示例

With CSS float, an element can be pushed to the left or right, allowing other elements to wrap around it.

Float is very often used for images, but it is also useful when working with layouts.

<img style="float:left;" src="../../images/float.gif"/>

/*浮动经常会用在图片上，同时也很频繁用在布局上，例如将block元素float在一行。*/

<img src="float_image1.jpg" style="float:left">
<img src="float_image3.jpg" style="float:right">

 

/*元素只能水平浮动，没有上下浮动

一个浮动的元素会尽可能地向父元素的左或向右挤推。

跟在浮动元素后面的元素会围绕浮动元素。

在浮动元素前面的元素不会受到浮动元素的影响。

如果一个图片右浮动，跟在图片后面的文本将会环绕在图片左边。*/
