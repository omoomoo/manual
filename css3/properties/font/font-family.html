font-family

语法

font-family：family-name generic-family

取值

取值 说明
family-name
generic-family		 A prioritized list of font family names and/or generic family names
inherit Specifies that the font family should be inherited from the parent element

说明

font-family属性为元素的字体指定字体类型。

font-family属性可以指定多组字体类型，如果第一个字体不可用，就会使用下个字体，以此类推。

有两种字体font-family的名字s:

family-name - The name of a font-family, like "times", "courier", "arial", etc.

generic-family - The name of a generic-family, like "serif", "sans-serif", "cursive", "fantasy", "monospace".

Start with the font you want, and always end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available.

Note: Separate each value with a comma.

Note: If a font name contains white-space, it must be quoted. Single quotes must be used when using the "style" attribute in HTML.

默认值 not specified
继承 yes
版本 css1
Javascript脚本 object.style.fontFamily="arial,sans-serif"

兼容性

IE FireFox Chrome Safari Opera
         

Note: The value "inherit" is not supported in IE7 and earlier. IE8 requires a !DOCTYPE. IE9 supports "inherit".
