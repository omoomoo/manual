font

语法

font：font-style font-weight font-size/line-height font-family

取值

取值 说明
font-style 指定字体样式。
font-variant 指定字体装换。
font-weight 指定字体粗细。
font-size/line-heigh 指定字体大小或行高。
font-family 指定字体类型。

说明

font属性是所有字体属性的缩写。The font shorthand property sets all the font properties in one declaration.

所有的字体属性可以按照以下方式排列：font-style font-variant font-weight font-size/line-height font-family

font-size和font-family的值是必需的。如果只指定一个，另外一个将使用默认值。

Note: The line-height property sets the space between lines.

默认值 not specified
继承 yes
版本 css1
Javascript脚本 object.style.font="italic small-caps bold 12px arial,sans-serif"

兼容性

IE FireFox Chrome Safari Opera
         

Note: The value "inherit" is not supported in IE7 and earlier. IE8 requires a !DOCTYPE. IE9 supports "inherit".
