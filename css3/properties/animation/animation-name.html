animation-name

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

animation-name：keyframename | none

取值

取值 说明
keyframename 指定关键帧的名字用来绑定到选择器上。
none 指定没有animation动画。

说明

animate-name由@keyframes指定。例如 @keyframes kf,这里的kf就是一个animate-name。

默认值 none
继承 no
版本 css3
Javascript脚本 object.style.animationName="mymove"
