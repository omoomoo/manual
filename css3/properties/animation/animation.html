animation

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

animation： animation-name animation-duration animation-timing-function animation-delay animation-iteration-count animation-direction.

取值

取值 说明
animation-name 指定关键帧用来绑定到选择器上。
animation-duration 指定动画的持续时间（秒或毫秒）。
animation-timing-function 指定动画的速度曲线函数。
animation-delay 指定动画的延迟执行时间。
animation-iteration-count 指定动画的循环次数。
animation-direction 指定动画是否反向交替播放。

说明

默认值 none 0 ease 0 1 normal
继承 no
版本 CSS3
Javascript脚本 object.style.animation="mymove 5s infinite"

兼容性

IE FireFox Chrome Safari Opera
  5      

示例

#demo1 {-webkit-animation:kf1 4s linear 0s infinite alternate;}

/*kf1为指定的keyframes，infinite指定动画无限循环播放，alternate指定动画反向交替播放*/


@-webkit-keyframes kf1 { /*keyframes指定关键帧，相当于flash的帧*/
25% {margin:0px 0px 0px 250px;}
50% {margin:150px 0px 0px 250px;}
75% {margin:150px 0px 0px 0px;}
100% {margin:0px;}
}

/*为之前的动画加入rotateY（）来实现3D动画。*/

#demo1 {-webkit-animation:kf2 4s linear 0s infinite alternate;}


@-webkit-keyframes kf2 { /*keyframes指定关键帧，相当于flash的帧*/
25% {margin:0px 0px 0px 250px;-webkit-transform:rotateY(360deg);}
50% {margin:150px 0px 0px 250px;-webkit-transform:rotateY(-360deg);}
75% {margin:150px 0px 0px 0px;-webkit-transform:rotateY(360deg);}
100% {margin:0px;-webkit-transform:rotateY(-360deg);}
}
