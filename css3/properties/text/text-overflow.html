text-overflow

语法

overflow：clip | ellipsis

取值

取值 说明
clip 剪切文本，讲溢出的部分文本剪切。
cllipsis 将溢出的部分文本用"..."代替。

说明

text-overflow属性用来指定当文本溢出父元素时该如何处理。

默认值 clip
继承 no
版本 css3
Javascript脚本 object.style.textOverflow="ellipsis"

兼容性

IE FireFox Chrome Safari Opera
5 7 4 4 10.6

示例

The text-overflow property specifies what should happen when text overflows the containing element.
