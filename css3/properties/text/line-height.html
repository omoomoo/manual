line-height

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

line-height：normal | number | length | percentage | inherit

取值

取值 说明
normal 设置使用默认的行高。
number 设置数值设置行号，所得的行高将为字体大小乘以numebr的值。即 line-height = font-size * numebr。
length 使用px，pt，cm，em来设置行高。
percentage 使用百分比设置行高，所得的行高为字体大小乘以percentage的值。即 line-height = font-size * percentage。
inherit 继承。

说明

默认值 normal
继承 yes
版本 css1
Javascript脚本 object.style.lineHeight="2"
