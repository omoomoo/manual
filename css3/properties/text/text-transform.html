text-transform

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

text-transform：none | capitalize | uppercase | lowercase | inherit

取值

取值 说明
none No capitalization. The text renders as it is. This is default
capitalize 把每个单词的首字母变成成大写。
uppercase 把所有的单词变成大写。
lowercase 把所有的单词变成小写。
inherit 继承。

说明

The text-transform property is supported in all major browsers.

Note: The value "inherit" is not supported in IE7 and earlier. IE8 requires a !DOCTYPE. IE9 supports "inherit".

默认值 none
继承 yes
版本 no
Javascript脚本 object.style.textTransform="uppercase"
