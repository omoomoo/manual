text-decoration

说明：本文档兼容性测试基础环境为：windows系统；IE6-IE10, Firefox9.0, Chrome16.0, Safari5.1, Opera11.60

语法

text-decoration：none | underline | overline | line-through | blink | inherit

取值

取值 说明
none 使用默认值。
underline 在文本下面添加下划线。
overline 在文本上面添加加上划线。
line-through 在文本中间添加删除线。
blink 设置文本闪烁。（ IE, Chrome,Safari不支持）
inherit 继承父元素的text-decoratiion的值。

说明

Note: The value "inherit" is not supported in IE7 and earlier. IE8 requires a !DOCTYPE. IE9 supports "inherit".

Note: The "blink" value is not supported in IE, Chrome, or Safari.

默认值 none
继承 no
版本 css1
Javascript脚本 object.style.textDecoration="overline"
