text-shadow

语法

text-shadow：h-shadow v-shadow blur color

取值

取值 说明
h-shadow 必选，水平阴影偏移值。可以为负值。
v-shadow 必选，垂直阴影偏移值。可以为负值。
blur 可选，阴影的模糊值。不允许为负值。默认为0。
color 可选，阴影的颜色。默认为黑色。

说明

The text-shadow property attaches one or more shadows to text. The property is a comma-separated list of shadows, each specified by 2 or 3 length values and an optional color. Omitted lengths are 0.

默认值 none
继承 no
版本 css3
Javascript脚本 object.style.textShadow="2px 2px #ff0000"

兼容性

IE FireFox Chrome Safari Opera
10 3 3 3 10
